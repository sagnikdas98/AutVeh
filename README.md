# Yonda
To build a 4 Wheel differential drive robot car.  
Hardware components :  
1. Nvidia Jetson
2. Raspberry pi 4
3. EPL Focal Camera 
4. Razor IMU 9DOF
5. Rp LiDAR A2
6. Cytron MD13S 13Amp DC Motor Driver
7. Ublox GPS module
