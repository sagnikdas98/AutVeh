# Yonda
To build a 4 Wheel differential drive robot car.  
Hardware components :  
1. Raspberry pi 3B+ 
2. 5MP Pi camera
3. OV7670 camera
4. MPU 6050
5. Arduino Uno R3 - 2x
6. L293D Motor Driver Shield for Arduino 
7. Measuring Sensor Groove Coupler Module For Arduino - 4x
8. 4 Wheel differential drive chasis 
9. 5000mAh Power Bank
10. 3 x 18650 + 3S BMS
